# 备份与恢复指南

rremind 默认使用浏览器 `localStorage` 保存数据，若需要在多台设备之间迁移或手动保存历史记录，请按照下面步骤操作。

## 备份（导出）

1. 在首页点击“订阅列表”右上角的 **备份数据**。
2. 浏览器会自动下载一个 `rremind-backup-YYYY-MM-DD.json` 文件。
3. 请妥善保存或同步到云盘。文件中包含订阅记录和 Telegram 设置等敏感信息，如需共享，请先手动编辑文件删除相关字段。

## 恢复（导入）

1. 在首页点击“恢复备份”按钮，选择之前下载的 JSON 文件。
2. 程序会校验格式并覆盖本地数据（包括设置、订阅、提醒状态）。
3. 导入成功后会重新渲染列表，如失败会显示错误提示。

> 提醒：恢复操作无法撤销，建议在导入前先执行一次备份。

## FAQ

- **可以跨浏览器吗？** 可以，只要浏览器支持 `localStorage` 即可（Chrome/Edge/Firefox/Safari）。
- **如何只迁移部分订阅？** 暂不支持，推荐导出后用文本编辑器删除不需要的条目，再导入。
- **备份加密吗？** 默认未加密，可结合操作系统自带的文件加密或密码压缩包保护。
